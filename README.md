# Laravel Skills

Repositorio personal de skills para trabajar con proyectos Laravel.

## Skills

- `laravel-action-pattern`: guia para aplicar clases Action en Laravel y mover logica de negocio fuera de controllers y models.

## Instalación
- Boost (recommended):
```bash
php artisan boost:add-skill JaviLerma/laravel-skills
```

- NPX
```bash
npx skills add JaviLerma/laravel-skills
```

## Codex plugin

Este repo tambien contiene una estructura de plugin Codex en la raiz, con el manifest en `.codex-plugin/plugin.json` y los skills del plugin en `skills/`.

La entrada de marketplace para instalarlo desde la UI de Codex queda para un paso posterior.
